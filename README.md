# Study_GDUT
### 简介

供软工三班同学食用，也可以发给其他班同学。每一个科目内含前人整理的资料以及各位同学收集到课内外的资料。如果有同学想将自己的资料分享给大家，可以联系学委📢

希望在剩下几周时间内，大家可以认真复习👍，❌拒绝摆烂

### 使用

* 第一次食用的话要现在本地克隆

```linux
git clone https://github.com/CodeGrindstone/Study_GDUT.git
```

* 克隆后，再命令行输入git pull就好啦
